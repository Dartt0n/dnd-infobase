|          |                                                                        |
| -------- | ---------------------------------------------------------------------- |
| Тип      | Превращение                                                            |
| Уровень  | Дрд 6, Жр 6, Маг/Чар 6                                                 |
| Диапазон | Ближний (7,5 м+1,5 м/2 ур.)                                            |
